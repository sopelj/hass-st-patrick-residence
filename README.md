# [St-Patrick Residence site Integration for Home Assistant](https://github.com/sopelj/hass-st-patrick-residence)

This is a very niche integration to allow for fetching, translating and displaying information for the internal web service of the St-Patrick's Residence in Québec.

I really just made this so my dad can easily see what meals are available for the current day in English in his Home Assistant. Maybe it can be a good example for similar integrations for others.
